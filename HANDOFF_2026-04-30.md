### New shared infrastructure (under `src/lib/`, `src/hooks/`)
- **`lib/storage.ts`** — `uploadMediaFile(file, folder)` returns Firebase Storage download URL. Folders: `products`, `posts`, `events`, `resources`, `media`.
- **`hooks/useFirestoreCollection.ts`** — Generic realtime CRUD hook with `where`/`orderBy`. Used by 6 features.
- **`lib/requireAuth.ts`** — `requireAuth(user, action)` gates an action behind login; if not signed in, redirects to `/client?returnTo=<currentPath>`. `consumeReturnTo()` is called inside `ClientPortal` after login.
- **`hooks/useCheckout.ts`** — `startCheckout({ purpose, lineItems, metadata })` calls the Cloud Function and redirects to Stripe.
- **`functions/src/createCheckoutSession.ts`** + **`functions/src/stripeWebhook.ts`** — generic, dispatches by `metadata.purpose` (order|event|class|subscription).

## What you need to do when you're back

### 1. Install new dependencies
```bash
cd "/Users/lesalondesinconnus/Documents/Websites/territoire-incarné (1)"
npm install
cd functions && npm install
```

This pulls in **`sharp`**, **`react-image-crop`** (frontend), and **`stripe`** (functions).

### 2. Deploy Firestore rules
```bash
firebase deploy --only firestore:rules
```

Rules now cover: `leads`, `products`, `resources`, `events/*/registrations`, `classes/*/requests`, `classes/*/members`, `classes/*/messages`, `posts`, `orders`, `users/*/subscriptions`, `notifications`.

### 3. Set Stripe secrets in Firebase
```bash
firebase functions:secrets:set STRIPE_SECRET_KEY
# Paste the test secret key (sk_test_...) when prompted

# After step 4 below:
firebase functions:secrets:set STRIPE_WEBHOOK_SECRET
```

### 4. Deploy functions
```bash
cd functions && npm run build && cd ..
firebase deploy --only functions
```

This deploys `createCheckoutSession`, `stripeWebhook`, and the existing `transcribeMeeting`. After deploy, copy the `stripeWebhook` URL.

### 5. Configure Stripe webhook (Stripe Dashboard)
1. Go to https://dashboard.stripe.com/test/webhooks (test mode!)
2. **Add endpoint** → paste the `stripeWebhook` function URL from step 4
3. Select events:
   - `checkout.session.completed`
   - `customer.subscription.updated`
   - `customer.subscription.deleted`
   - `invoice.payment_failed`
4. Reveal the **Signing secret** (`whsec_...`) → run `firebase functions:secrets:set STRIPE_WEBHOOK_SECRET`
5. Re-deploy functions: `firebase deploy --only functions`
